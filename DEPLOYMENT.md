# 🚀 הוראות העלאה ל-GitHub Pages

## ✅ קבצים שצריך להעלות ל-GitHub

וודאי שיש לך את הקבצים האלה בריפו:

1. ✅ `index.html` - דף הבית
2. ✅ `salary-calculator.jsx` - המחשבון
3. ✅ `README.md` - הוראות עדכון (אופציונלי)

---

## 📤 שלב 1: העלאת הקבצים ל-GitHub

אם עדיין לא העלית את הקובץ `index.html`:

### דרך א': דרך האתר (קל!)

1. היכנסי לריפו שלך ב-GitHub
2. לחצי על **"Add file"** → **"Upload files"**
3. גררי את הקובץ `index.html` 
4. לחצי **"Commit changes"**

### דרך ב': דרך שורת הפקודה

```bash
# אם את בתיקיית הפרויקט
git add index.html
git commit -m "Add index.html for GitHub Pages"
git push
```

---

## 🌐 שלב 2: הפעלת GitHub Pages

1. **היכנסי לריפו שלך ב-GitHub**
   
2. **לחצי על "Settings"** (למעלה, בשורת התפריט)
   
3. **בתפריט הצדי השמאלי:**
   - גללי למטה למקטע **"Code and automation"**
   - לחצי על **"Pages"**

4. **הגדרות Source:**
   - תחת **"Build and deployment"**
   - ב-**"Source"**: בחרי **"Deploy from a branch"**
   - ב-**"Branch"**: 
     - בחרי **"main"** (או "master" אם זה שם הענף שלך)
     - בחרי **"/ (root)"**
   - לחצי **"Save"**

5. **המתיני כמה דקות** ⏳
   - GitHub יבנה את האתר
   - תראי הודעה ירוקה עם הקישור

---

## 🔍 איך לבדוק שזה עובד?

1. **חזרי לעמוד "Pages" ב-Settings**
2. **בראש העמוד** תראי הודעה:
   - ✅ ירוק: "Your site is live at https://..."
   - 🟡 צהוב: "GitHub Pages is currently building..."
   - ❌ אדום: יש בעיה - ראי למטה

3. **לחצי על הקישור** וודאי שהמחשבון עובד!

## ❗ פתרון בעיות נפוצות

### בעיה 1: "404 - Page not found"

**פתרון:**
- ✅ וודאי שיש קובץ בשם `index.html` (עם i קטנה!)
- ✅ וודאי שהקובץ נמצא בשורש הריפו (לא בתיקייה)
- ✅ המתיני 5-10 דקות נוספות

### בעיה 2: "Error: The page build failed"

**פתרון:**
- ✅ וודאי שיש קובץ `index.html`
- ✅ בדקי שהקובץ תקין (אין שגיאות סינטקס)
- ✅ נסי לעשות commit חדש:
  ```bash
  git commit --allow-empty -m "Trigger rebuild"
  git push
  ```

### בעיה 3: העמוד נטען אבל לא רואים את המחשבון

**פתרון:**
- ✅ וודאי ששני הקבצים (`index.html` ו-`salary-calculator.jsx`) באותה תיקייה
- ✅ פתחי את הקונסול בדפדפן (F12) וחפשי שגיאות
- ✅ וודאי שהקובץ `salary-calculator.jsx` מתחיל ב-`/* global React, ReactDOM */`

### בעיה 4: לא רואה את אפשרות "Pages" ב-Settings

**פתרון:**
- ✅ וודאי שהריפו הוא **Public** (לא Private)
- ✅ אם הריפו Private, שני אותו ל-Public:
  - Settings → Danger Zone → Change visibility → Make public

## 🔄 עדכון האתר אחרי שינויים

כשאת משני משהו בקוד:

1. **העלי את הקובץ המעודכן ל-GitHub**
   ```bash
   git add .
   git commit -m "עדכון מחשבון"
   git push
   ```

2. **GitHub Pages יעדכן אוטומטי** תוך 1-5 דקות

3. **לפעמים צריך לרענן עם Cache:**
   - Windows: `Ctrl + Shift + R`
   - Mac: `Cmd + Shift + R`

   - פתחי את האתר
